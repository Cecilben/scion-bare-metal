Source code of the KNET test application

Steps to compile using the makefiles in src/knet/arm and src/knet/x86:
0. Clone the OpenNSL repository and check out commit eb1cfd0.
1. Copy the src folder into the OpenNSL root directory.
2. Copy the library binary libopennsl.so.1 into to bin/as4610 or bin/as5812 under the OpenNSL root directory.
3. Invoke the makefiles.
