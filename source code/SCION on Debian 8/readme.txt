Script to install SCION on Debian 8

Steps to install SCION on Debian 8:
1. Run install_scion.sh
2. Configure Zookeeper as in scion_install_script.sh[1] after the line "bash -c 'yes | GO_INSTALL=true ./env/deps'"

[1] https://raw.githubusercontent.com/netsec-ethz/scion-coord/master/scion_install_script.sh
