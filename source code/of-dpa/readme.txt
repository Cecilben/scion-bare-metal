Source code of the OF-DPA test application

Steps to compile using the makefiles in src/ofdpa_client/arm and src/ofdpa_client/x86:
0. Clone the OF-DPA repository and check out commit 7c4b5f2.
1. Merge the src folder into the src folder of the OF-DPA repository.
2. Copy the library binary librpc_client.so.1 into to bin/as4610 or bin/as5812 under the OF-DPA root directory.
3. Invoke the makefiles.
